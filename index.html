Projects

CropXpert

CropXpert is a smart soil analysis system that measures soil parameters such as NPK nutrients, pH level, temperature and moisture using sensors. The collected data is processed and used in a machine learning model that recommends the most suitable crop for the soil.

RehabMate

RehabMate is an AI-powered physiotherapy assistant that helps users perform exercises like pushups and squats correctly. The system uses MediaPipe pose detection and a camera to analyze body posture while a motorized scissor lift provides real-time physical support based on movement feedback.
